## Usage Instructions

1. Fill in the Header section before or during the lecture/reading.
2. During the session, focus on the Main Notes column — capture key points, not every word.
3. Within 24 hours, add Cue Questions in the left column that test your understanding of each section.
4. Write the Summary section from memory, then check against your notes.
5. Fill in Key Terms, Connections, and Follow-Up Actions.
6. Use the Cue Column to self-test: cover the Main Notes and try to answer each question.
